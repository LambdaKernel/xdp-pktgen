# xdp-pktgen

## Basic usage

### You will need to Go compiler
Get it from [Golang website](https://go.dev/dl/) and install it

Build the code

```bash
    ./build
```

Run the program

```bash
    sudo ./xdp-pktgen -v -p <dst port> <hostname or ip>
```

OBS.: It's needed to run the binary with privileges
